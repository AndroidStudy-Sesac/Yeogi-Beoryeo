## [Unreleased]

## [0.2.0] - 2026-07-24

### Added

- 최초 실행에서 핵심 기능 6단계를 안내하고 설정에서 다시 실행할 수 있는 앱 사용 가이드
- 지도와 외부 링크 실행 전에 인터넷, 정확한 위치 권한, 기기 위치 기능을 안내하고 복구 후 원래 요청을 이어가는 흐름
- 수거 장소가 100개 이상이면 마커를 묶어 표시하고, 선택한 카테고리 결과가 없으면 전체 장소로 돌아갈 수 있는 지도 기능
- 법정동·관리구역명·대상지역명 별칭으로 지역 가이드를 찾고 실제 제공 가능한 공식 지역만 보여주는 검색

### Changed

- 지역 가이드의 검색, 지역 선택, 후보, 오류 화면을 가로 화면과 짧은 화면에서도 조작할 수 있도록 개선
- 지도 바텀시트 높이를 콘텐츠, 글자 크기, 화면 크기, 시스템 영역에 맞추고 하단 액션과 NAVER 지도 로고가 가려지지 않도록 개선
- 하단 탭으로 돌아오면 이전 화면 상태를 복원하고 현재 탭을 다시 누르면 첫 화면으로 이동하도록 탐색 정책 정리
- 개발용 앱의 패키지, 이름, Firebase 클라이언트를 배포용 앱과 분리해 같은 기기에 함께 설치할 수 있도록 변경
- CI에서 전체 Android lint, 기기 테스트, R8 release APK, Play 제출용 AAB와 bundletool 검증, 지역 가이드 제공 지역 검증을 실행하도록 출시 경로 강화
- 기존 데이터베이스의 v1·v2에서 v3으로 업데이트할 때 즐겨찾기와 장소 정보가 유지되는지 검증하도록 보강

### Fixed

- 빠른 카테고리 설정에서 돌아온 뒤 펼친 목록의 접기 버튼이 화면 밖에 남는 문제를 수정하고 최대 선택 개수 안내 추가
- 글자 크기 200%에서 홈 안내 배너와 설정 상세 제목이 잘리는 문제 수정
- 품목 검색 취소·뒤로가기·연속 입력 뒤 상태가 어긋나는 문제와 같은 품목 상세를 다시 열 때 불필요한 로딩이 발생하는 문제 수정
- 지역 가이드에서 같은 일정 후보가 중복 노출되거나 후보 복귀 뒤 검색·선택 상태가 남는 문제 수정
- 공공데이터에 없는 대형폐기물 배출 요일을 임의로 표시하지 않고 잘못 연결된 교차 시군구 행정동 매핑 제거

### Security

- CI release 검증에서 실제 업로드 키와 서비스 Firebase 설정을 사용하지 않고 임시 서명 키와 더미 설정만 사용하도록 분리

## [0.1.1] - 2026-07-16

### Added

- 저장한 지역 가이드 중 홈에 표시할 대표 지역을 직접 고정하는 기능
- 설정의 출처 및 이용조건에서 NAVER 지도 법적 고지와 오픈소스 라이선스를 확인하는 경로

### Changed

- 홈 지역 가이드 대표 지역과 이전 요약을 앱 재실행, 재수집, 로딩, 조회 실패 상황에서도 안정적으로 유지
- 저장한 지역 가이드의 긴 지역명을 최대 2줄로 표시하고 대표 지역 고정 버튼의 터치 영역과 TalkBack 설명 개선
- CI의 기본 검증과 전체 검증에 app 모듈 단위 테스트를 추가하고 `main` 대상 release PR도 검증하도록 보강

### Fixed

- 지역 선택 중 검색 후보 목록과 지역 선택 목록이 동시에 노출되는 문제 수정
- 지역 가이드 후보 상세에서 돌아오면 후보 목록의 스크롤 위치가 초기화되는 문제 수정
- 동일한 지역 가이드 후보가 여러 개 내려오면 날짜 정보가 유효한 최신 항목을 우선 적용해 중복 노출을 줄임
- NAVER 지도 로고 클릭으로 비정상 종료될 수 있는 경로를 차단하고 바텀시트에 로고가 가려지는 문제 수정
- release 테스트와 lint까지 키스토어를 요구하던 서명 검증을 실제 배포 산출물 작업에만 적용하도록 수정

## [0.1.0] - 2026-07-14

### Added

- 품목 검색과 품목별 분리배출 방법, 특징, 주의사항 안내
- 현재 위치와 지역 검색을 이용한 주변 수거 장소 지도 조회
- 지역별 배출 요일, 시간, 장소 안내
- 품목, 수거 장소, 지역 가이드 즐겨찾기
- 앱 출처와 개인정보처리방침 확인
- Firebase Crashlytics를 이용한 비정상 종료 및 오류 진단

### Changed

- Google Play 배포 빌드에 R8 코드 최적화, 난독화, 미사용 리소스 축소 적용
- 앱의 출처 및 이용조건 화면에 정부 정보 공식 출처 링크와 정부기관 비제휴 안내 추가

### Security

- 저장소 밖의 업로드 키로 서명된 Google Play 배포용 AAB 생성 절차
