# Data Sources

이 문서는 앱과 공개 저장소에 포함된 외부 가공 데이터의 출처, 기준일, 가공 여부를 정리합니다.

## 기후에너지환경부 분리배출 데이터

- 데이터명: 생활폐기물 분리배출 누리집 품목 및 대표 가이드 데이터
- 제공기관: 기후에너지환경부
- 원본 자료:
  - [생활폐기물 분리배출 누리집](https://xn--oy2b29bd3a601b.kr/)
  - [기후에너지환경부 분리배출 정보조회 서비스](https://www.data.go.kr/data/15156866/openapi.do)
- 데이터 확인일: 2026-05-31
- 이용조건 확인일: 2026-07-13
- 가공 파일:
  - `item_disposal_guides.json`
  - `representative_guide_details.json`
- 앱 내 사용 목적:
  - 품목별 분리배출 방법 검색 및 상세 안내
  - 대표 분류별 배출 방법 안내
- 가공 여부: 품목명, 유사 품목, 분류 경로, 배출 방법, 특징, 주의사항 등 텍스트 정보를 앱 사용 목적에 맞게 JSON 형태로 가공
- 이미지 포함 여부: 누리집 이미지 파일은 포함하지 않음
- 이용허락 범위: 공공데이터포털 안내 기준 제한 없음
- 앱 asset 포함: 공식 답변을 기준으로 가능하다고 판단
- 공개 GitHub 저장소 게시: 공식 답변을 기준으로 가능하다고 판단
- Google Play 배포: 공식 답변을 기준으로 가능하다고 판단
- 관련 이슈: #168, #85, #45

기후에너지환경부는 재활용 가능 자원의 분리배출 방법 등을 공공데이터로 공개하고 있으며, 공공데이터법에서 정한 제한 사유가 없다면 자유롭게 이용할 수 있다고 답변했습니다. `기후에너지환경부_분리배출 정보조회 서비스`의 이용이 가능하다는 안내와 공공데이터포털의 이용허락 범위가 제한 없음인 점을 근거로 위 가공 데이터를 유지합니다.

출처 표시 문구:

> 기후에너지환경부 생활폐기물 분리배출 누리집과 분리배출 정보조회 서비스의 텍스트 정보를 앱 사용 목적에 맞게 가공한 데이터입니다.

주의사항:

- 원본 자료의 출처, 데이터 확인일, 이용조건 확인일, 가공 여부를 함께 표시합니다.
- 품목명과 분류 경로 등은 정책 변화에 따라 변경될 수 있습니다.
- 데이터 갱신 시 확인일과 변경 내용을 함께 갱신해야 합니다.

## 행정안전부 지역별 배출 데이터

- 데이터명: 행정안전부 생활쓰레기배출정보 조회서비스
- 제공기관: 행정안전부
- 원본 자료: [행정안전부 생활쓰레기배출정보 조회서비스](https://www.data.go.kr/data/15155080/openapi.do)
- 데이터 확인일: API 조회 시점
- 업데이트 주기: 일간
- 앱 내 사용 목적:
  - 지역별 생활쓰레기, 음식물쓰레기, 재활용품 배출 방법 안내
  - 지역별 배출 요일, 시간, 장소 안내
- 가공 여부: API 응답을 앱의 지역별 배출 가이드 모델로 변환
- 이용허락 범위: 공공데이터포털 안내 기준 제한 없음
- 관련 이슈: #168

공공데이터포털의 Open API를 조회해 지역별 배출 정보를 제공합니다. API 응답 원본을 앱 asset이나 저장소 파일로 포함하지 않습니다.

## 행정안전부 행정구역 데이터

- 데이터명: 행정안전부 행정구역 데이터
- 제공기관: 행정안전부
- 원본 자료: 행정기관 및 관할구역 코드 자료
- 원본 파일:
  - `KIKcd_H.20260701.xlsx`
  - `KIKcd_B.20260701.xlsx`
  - `KIKmix.20260701.xlsx`
- 데이터 기준일: 2026-07-01
- 가공 파일:
  - `administrative_regions.20260701.json`
  - `legal_to_admin_mappings.20260701.json`
  - `regional_guide_regions.20260701.json`
- 앱 내 사용 목적:
  - 지역 선택
  - 법정동-행정동 매핑
  - 지역 가이드 후보 선택 보조
- 가공 여부: 원본 자료를 앱 사용 목적에 맞게 JSON 형태로 가공
- 앱 asset 포함: 답변 내용을 기준으로 가능하다고 판단
- 공개 GitHub 저장소 게시: 답변 내용을 기준으로 가능하다고 판단
- Google Play 배포: 답변 내용을 기준으로 가능하다고 판단
- 관련 이슈: #168, #155

행정안전부 홈페이지에 공개된 행정기관 및 관할구역 코드 자료를 기준으로, 앱에서 지역 선택과 지역별 배출 가이드 후보 선택을 보조할 수 있도록 JSON 형태로 가공했습니다.

출처 표시 문구:

> 행정안전부 행정기관 및 관할구역 코드 자료를 기준으로 앱 사용 목적에 맞게 가공한 데이터입니다. 데이터 기준일은 2026-07-01입니다.

주의사항:

- 원본 자료의 출처, 기준일, 가공 여부를 함께 표시합니다.
- 공개 저장소에 불필요한 식별 정보는 포함하지 않습니다.
- 행정구역 변경이 발생할 수 있으므로, 데이터 갱신 시 기준일과 가공 파일명을 함께 갱신해야 합니다.
